O que são as DIV e SPAN?

No HTML, <div> e <span> são elementos de marcação amplamente utilizados para estruturar e estilizar o conteúdo de uma página. Embora ambos sejam "contêineres" de conteúdo, eles têm propósitos e características distintas.

  1. <div> (Divisão)
    • Bloco de nível: O elemento <div> é um elemento de "nível de bloco" (block-level), o que significa que ele ocupa a largura total de seu contêiner pai. Isso cria uma quebra de linha antes e depois do elemento, isolando-o visualmente de outros conteúdos.
    • Estruturação de Layout: É usado para agrupar elementos relacionados, como seções, colunas e blocos de conteúdo maiores. Frequentemente, ele é usado em conjunto com CSS para criar layouts, pois permite organizar diferentes partes da página
    • Exemplo de uso: 
      
<div class="container">
    <h1>Título da Seção<h1>
    </p>Parágrafo dentro da div.</p>
    • Quando usar: Utilize <div> quando quiser agrupar elementos para criar uma seção maior ou dividir a página em blocos maiores.
  2. <span> (Linha)
    • Linha de nível (inline): O elemento <span> é de "nível de linha" (inline-level), ou seja, ele não quebra a linha e ocupa apenas o espaço necessário para o seu conteúdo. Pode ser usado dentro de parágrafos ou outros elementos de linha, sem afetar o layout geral.
    • Aplicação de Estilos Específicos: Geralmente, é usado para aplicar estilos ou identificadores específicos a uma parte menor do conteúdo, como uma palavra ou frase dentro de um parágrafo.
    • Exemplo de uso: 
      
<p>Este é um texto com uma <span classe="destaque">palavra destacada<span> usando span.<p>
    • Quando usar: O <span> é ideal para estilizar ou aplicar comportamentos específicos a uma pequena parte do conteúdo sem afetar o layout.
